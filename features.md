# Project Features

This document outlines the core features of the Household Management application and the status of their test coverage.

## Feature List

*   **Authentication:** Manages user sign-up, login, logout, and session management.
    *   **Test Status:** Tested.
*   **Plant Management:** Handles creating, updating, deleting, and listing household plants.
    *   **Test Status:** Tested.
*   **Project Management:** Manages larger household projects, including tasks and progress tracking.
    *   **Test Status:** Tested.
*   **Task Management:** A simpler system for managing individual tasks.
    *   **Test Status:** Tested.
*   **Calendar:** Provides a calendar view for tasks and events.
    *   **Test Status:** Tested.
